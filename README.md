# Wings Cafe

React + Express app for cafe management.

## Setup
1. `npm install`
2. `npm run build`
3. `npm run server`
4. Visit http://localhost:10000

## Render Deploy
- Build: `npm install && npm run build`
- Start: `node server/server.js`
- Env: NODE_ENV=production

## API
- GET /api/products
- GET /api/transactions
